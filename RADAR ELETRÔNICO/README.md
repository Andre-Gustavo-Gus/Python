# Radar Eletrônico
 
Este é um script simples em Python que simula um radar eletrônico. Ele verifica se a velocidade informada excede o limite de 80 km/h e, caso ultrapasse, calcula e exibe o valor da multa.

 Funcionalidade
* Detecta se o motorista ultrapassou o limite de 80 km/h.

* Aplica uma multa de R$7,00 por km excedido.

* Informa o valor da multa ou parabeniza por estar dentro do limite.

* Exibe mensagens com cores no terminal (em terminais que suportam ANSI escape codes).
